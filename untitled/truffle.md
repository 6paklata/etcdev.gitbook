# Getting Starterd

## Install Dependencies 

**Truffle**

```text
$ npm install -g truffle
```

Confirm truffle is installed

```text
$ truffle version
Truffle v4.1.13 (core: 4.1.13)
Solidity v0.4.24 (solc-js)
```
